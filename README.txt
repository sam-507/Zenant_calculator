# Zenant Zero Deposit Security Loan Calculator

A web-based calculator for computing EMI and loan details for Zenant's zero deposit security loan service.

## Features

- Calculate required security deposit based on monthly rent
- Compute EMI based on loan amount and tenure
- View detailed amortization schedule
- Interactive pie chart showing principal vs interest breakdown
- Responsive design that works on all devices

## Setup

1. Clone the repository
2. Open `index.html` in a web browser
3. No build process required - it's pure HTML, CSS, and JavaScript

## Embedding

To embed this calculator in another website:

1. Host these files on a web server
2. Use an iframe to embed:
```html
<iframe src="YOUR_HOSTED_URL" width="100%" height="1200" style="border: none; width: 100%;"></iframe>
```

## Dependencies

- Chart.js (loaded via CDN)

## Structure

- `index.html` - Main HTML file
- `assets/css/styles.css` - Stylesheet
- `assets/js/calculator.js` - Calculator logic
